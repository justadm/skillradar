# План развития GridAI

## 1) Качество выдачи
- улучшить ранжирование (вес зарплаты/навыков/опыта);
- синонимы ролей (backend → server‑side, QA → quality);
- фильтрация по реальному набору навыков.

## 2) B2B‑функции
- отчеты по рынку (PDF/CSV);
- тренды за 7/30 дней;
- сегментация по регионам/форматам.

## 3) Автоматизация
- расписание отчетов HR;
- сохранённые “профили роли”;
- уведомления о трендах.
