# Roadmap

Приоритет: **Telegram‑MVP**. Рынок РФ, минимальный бюджет LLM.

## Сейчас (MVP)

### Бот и UX
- [ ] Команда `/start` и главное меню.
- [ ] Сценарий «Подбор вакансий».
- [ ] Сценарий «Рынок навыков».
- [ ] Управление стоп‑листом (добавить/удалить).

### Интеграции
- [ ] HH API клиент (vacancies, vacancy details).
- [ ] OpenAI клиент (парсер + объяснение + рынок).

### Данные
- [ ] SQLite схема и репозитории.
- [ ] Кэш вакансий и market‑snapshots.

### Логика
- [ ] Парсинг критериев и маппинг в фильтры HH.
- [ ] Скоринг и ранжирование.
- [ ] Фильтрация по стоп‑листу.

---

## Детальный план MVP (итерации)

1. [x] Документация: подробный план + ответы для формы регистрации приложения hh.ru.
2. [x] LLM‑кэш: парсинг критериев + пояснения по вакансиям.
3. [x] Бот‑health: `/status`, базовые логи и обработка ошибок.
4. [x] Локальные тест‑сценарии (ручные чек‑кейсы) и quickstart‑памятка.
5. [x] Автосоздание `.env` при старте, если отсутствует.
6. [x] Команда `/reset` для сброса состояния пользователя.
7. [x] Мини‑панель админа: список последних запросов.
