## B2B‑направление (план)

ЦА: HR/рекрутеры. Канал: TG. Данные: только публичные вакансии. Аналитика: базовая.

### Этап 1 — B2B‑MVP (TG)
- [x] Режим «HR‑аналитика»: переключатель в меню (B2B/Соискатель).
- [x] Команда «Рынок роли» → базовые метрики: объем, топ‑навыки, вилки, удаленка.
- [x] Команда «Конкуренты»: поиск по ключевым навыкам + топ работодателей.
- [x] Команда «Вакансия‑шаблон»: генерация требований/скиллов/вилок на основе рынка.
- [x] Отчет «одним сообщением» + PDF/файл по запросу (простая выгрузка).

### Этап 2 — Качество и сегментация
- [x] Сегменты по городам/удаленке/уровню.
- [x] Сравнение «прошлый период vs текущий» (тренд).
- [x] Фильтры по индустрии/типу компании (если доступно).

### Этап 3 — Монетизация
- [x] Лимиты в TG: X отчетов/день.
- [x] Тарифы: per‑seat подписка.
- [x] Paywall (ручной, потом автоматизация).

### Этап 4 — Расширение данных (если HH разрешит)
- [ ] Подключение резюме/откликов.
- [ ] Сопоставление спроса/предложения.
- [ ] Расширенные отчеты для HR.
